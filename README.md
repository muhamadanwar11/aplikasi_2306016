# Tugas 2 - Aplikasi Flutter Sederhana

## Deskripsi

Project ini merupakan **Tugas 2** pembuatan aplikasi sederhana menggunakan framework Flutter.
Aplikasi ini menampilkan halaman utama yang berisi artikel dan daftar komentar.

Aplikasi dibuat menggunakan bahasa pemrograman Dart dan menggunakan komponen UI dari Flutter seperti AppBar, Card, ListView, dan Image.

## Tampilan Aplikasi

Aplikasi memiliki beberapa bagian utama:

1. **AppBar**

   * Icon Home
   * Judul aplikasi: *Aplikasi Pertamaku*

2. **Artikel**

   * Gambar dari internet
   * Judul artikel
   * Deskripsi artikel

3. **Komentar**

   * Daftar komentar menggunakan ListView
   * Berisi beberapa pengguna seperti:

     * John Doe
     * Mary Doe
     * Alex Doe
     * Jane Doe
     * Michael Doe

---

## Fitur Aplikasi

* Menampilkan artikel sederhana
* Menampilkan gambar dari internet
* Menampilkan daftar komentar
* Tampilan menggunakan Card agar lebih rapi

## Kesimpulan

Aplikasi ini merupakan latihan dasar penggunaan Flutter untuk memahami struktur layout, penggunaan widget, serta pengelolaan tampilan menggunakan Column, Row, dan ListView.
